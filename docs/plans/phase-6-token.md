# Phase 6 — Token ($AGENT via Genesis)

**Goal:** Agent A wallet launch $AGENT token via Metaplex Genesis. Token = kepemilikan protokol, revenue share dari protocol fee.

## Model
- Token dilaunched oleh Agent A Asset Signer wallet
- Genesis Launch Pool: deposit SOL → dapat $AGENT proporsional
- Protocol fee (kecil) dari setiap request → accumulate → distribute ke $AGENT holders
